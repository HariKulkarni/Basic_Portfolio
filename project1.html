Musical Chord Identification

This project focuses on identifying piano chords using Machine Learning and Deep Learning techniques.

Technologies Used

  • Python
  • Machine Learning
  • Deep Learning (CNN)
  • NumPy, OpenCV

Project Description

Audio recordings were converted into waveform images using Discrete Fourier Transform (DFT). Deep learning models were trained and evaluated using accuracy, precision, recall, and F1-score.
